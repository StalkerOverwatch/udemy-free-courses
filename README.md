# Contributions

Do you want to contribute to the project? Follow the steps below:

```bash
$ pipenv install
```
and
```bash
$ pipenv install --dev
```
Run the virtual environment with:

```bash
$ pipenv shell
```

Create the database:

```bash
$ python manage.py migrate
```
And run the project on localhost:8000

```bash
$ python manage.py runserver
```

# Dependencies
