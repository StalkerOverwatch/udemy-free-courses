# Contributions

Do you want to contribute to the project? follow the steps below:

```python
$ pipenv install
```
and
```python
$ pipenv install --dev
```
Run the virtual environment with:

```python
$ pipenv shell
```

Create the database:

```bash
$ python manage.py migrate
```
And run the project on localhost:8000

```bash
$ python manage.py runserver
```

# Dependencies
